# Changelog

All notable changes to Mini CRM will be documented in this file.
Format based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-03-24

### Added
- Initial project structure with src, docs, tests, assets directories
- Contact management interface (HTML, CSS, JavaScript)
- Add contact form with name, email, phone, company fields
- Contact card grid with responsive layout
- Real-time search and filter functionality
- Delete contact with confirmation dialog
- localStorage persistence for contacts data
- Input validation for required fields

### Fixed
- Empty fields validation in contact form
- DOM rendering optimization for large contact lists
