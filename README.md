# Mini CRM – Gestionar de Contacte

## Descriere

Mini CRM este o aplicatie web simpla pentru gestionarea contactelor de afaceri. Permite adaugarea, vizualizarea, cautarea si stergerea contactelor, cu persistenta datelor in localStorage.

## Tehnologii utilizate

- HTML5
- CSS3 (Flexbox, CSS Grid, CSS Variables)
- JavaScript (ES6+, localStorage API)

## Functionalitati

- Adaugare contacte (nume, email, telefon, companie)
- Afisare contacte in grid responsive
- Cautare in timp real dupa nume, email sau companie
- Stergere contact cu confirmare
- Persistenta datelor in browser (localStorage)

## Instructiuni de rulare

1. Clonati repository-ul
2. Deschideti `src/index.html` intr-un browser modern
3. Nu este necesara nicio instalare sau configurare suplimentara
